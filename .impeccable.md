# Telos Design Context

## Users

求职者(中文为主,兼顾中英双语投递场景)。他们在求职高压期使用 Telos,目标是快速产出一份能通过 ATS、且让 HR 一眼认可的专业简历。使用场景多为桌面端深度编辑(填经历、调排版),辅以移动端查看/微调。核心情绪:时间紧、怕被筛掉、希望"有人帮我写好"。

## Brand Personality

**智能、高效、省心。** 不是温暖陪伴型,而是"临床级专业能力"——AI 在背后默默把重活干完,用户只管输入,产出即专业。语气直接、克制、不啰嗦;界面干净利落,每个元素都有存在的理由。让用户觉得"这个工具懂我、替我省了事"。

3 词:智能(Intelligent)、高效(Efficient)、省心(Effortless)。

## Aesthetic Direction

**专业商务 SaaS,精致打磨型。** 保留当前品牌蓝 + 灰阶底色,向 Notion / Linear / Vercel 的克制精致看齐:

- **Light theme 为主**——简历编辑是日间深度工作场景,浅色降低疲劳、强化"纸面"专业感。
- **内容优先**——简历内容是主角,UI chrome 退后;大量留白、弱化装饰。
- **精准排版**——Linear 式的像素级精度:对齐严谨、字号克制有层次、间距有节奏。
- **克制的智能感**——AI 能力在需要时浮现(选中文字才出现润色、粘贴 JD 才触发匹配),不做炫技动效。
- **安静的自信**——专业但不无聊;通过排版张力(大标题、衬线点缀)和细腻交互(hover、focus 过渡)传递质感,而非渐变/发光/花哨配色。

参考:Notion(内容优先、中性克制)、Linear(像素级精度、subtle 深色点缀)、Vercel(几何感、高对比、极简)。

Anti-reference:过度装饰的"AI 噱头"风(渐变发光、毛玻璃泛滥、彩色 blob 背景)、低质感的传统简历站(拥挤、廉价模板堆砌)。

## Design Principles

1. **Content is the hero** —— 简历内容永远是主角。UI 框架、导航、装饰元素都要为内容让路,绝不喧宾夺主。
2. **Pixel-precise restraint** —— 每个间距、对齐、字号都有理由。宁可少做,不可多饰。向 Linear 的精度看齐。
3. **AI surfaces contextually, never as gimmick** —— AI 能力在用户需要的那一刻才出现(选中文本→润色;粘贴 JD→匹配)。不做常驻悬浮、不做炫耀式动效。
4. **Speed is a feature** —— 交互必须感觉即时:乐观更新、流畅过渡、无阻塞。省心 = 不用等。
5. **Quiet confidence over loud decoration** —— 用排版张力(对比、层次)和细腻质感(hover、focus、微动效)传递专业,而非渐变、发光、彩色装饰。
